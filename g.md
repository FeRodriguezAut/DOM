Comprendiendo antes de programar

**Crear una tarea** `POST`Se usa para enviar datos al servidor y crear un nuevo recurso.**Listar tareas** `GET`Se usa para solicitar una representación de uno o varios recursos.**Actualizar una tarea** `PUT` / `PATCH``PUT` reemplaza la tarea completa; `PATCH` aplica modificaciones parciales.**Eliminar una tarea** `DELETE`Se usa para dar de baja un recurso específico.
